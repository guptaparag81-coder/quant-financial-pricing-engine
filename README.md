# Quantitative Financial Pricing Engine

A Python implementation of the Black-Scholes option pricing model for European call options.

## Objective

This project demonstrates how mathematical finance can be used to estimate the fair value of financial derivatives.

The implementation calculates European call option prices using the Black-Scholes model and visualises how option values vary with the underlying stock price.

## Features

- Black-Scholes call option pricing
- Numerical computation using NumPy
- Statistical functions using SciPy
- Option-price visualisation using Matplotlib
- Financial modelling and quantitative analysis

## Technologies

- Python
- NumPy
- SciPy
- Matplotlib

## Mathematical Background

The Black-Scholes model is one of the most widely used models in quantitative finance for pricing options and other financial derivatives.
